Examples
========

The examples below are rendered with all outputs on this page.  To **run them
locally**, you need `parallelproj installed
<https://parallelproj.readthedocs.io/en/stable/installation.html>`_ and the
single helper module below.  Download it into the **same folder** as the
example script — no extra configuration needed.

- `example_utils.py <https://raw.githubusercontent.com/KUL-recon-lab/parallelproj/main/docs/examples/example_utils.py>`_

Then run the example as usual::

   python 01_run_my_example.py
