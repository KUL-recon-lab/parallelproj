Examples
========

The examples below are rendered with all outputs on this page.  To **run them
locally**, you need `parallelproj installed
<https://parallelproj.readthedocs.io/en/stable/installation.html>`_ and the
three helper modules below.  Download them into the **same folder** as the
example script — no extra configuration needed.

- `array_utils.py <https://raw.githubusercontent.com/KUL-recon-lab/parallelproj/main/docs/examples/array_utils.py>`_
- `img.py <https://raw.githubusercontent.com/KUL-recon-lab/parallelproj/main/docs/examples/img.py>`_
- `vis.py <https://raw.githubusercontent.com/KUL-recon-lab/parallelproj/main/docs/examples/vis.py>`_

Then run the example as usual::

   python 01_run_my_example.py
